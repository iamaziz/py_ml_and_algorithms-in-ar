# الخوارزميات وتعليم الآلة باللغة العربية

 مرجع تعليمي وعملي مختصر لمواضيع في لغة بايثون، تعليم الآلة، خوارزميات الكمبيوتر، وهياكل البيانات.


 نرحب بكل المشاركين لإثراء المحتوى وإضافة المزيد من المواضيع.
